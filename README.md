# Документация Service for Label Creation


**SLC (Service for Label Creation)** — ***система***, предназначенная для формирования ярлыков и настройки правил формирования ярлыков. Для формирования ярлыков используется портал размещённый по адресу http://slc/  (физический адрес http://rnd-label/). Также, портал выполняет функцию передачи/заполнения дополнительных данных. Для редактирования правил формирования и шаблонов печати ярлыков используется отдельное локальное приложение, редактор **SLC Template Editor**.
На основании созданных и сохранённых в хранилище сервера правил осуществляется выбор шаблона печати для каждого отдельного размера товара и формируется ярлык по шаблону с подставленными данными из базы 1С: Справочники. Для печати каждого размера (т.е. каждого товара) происходит отдельный поиск шаблона по правилам и формирование ярлыка по найденному шаблону.

## Основные модули системы.

* Клиентская часть для редактирования. Локальное приложение **SLC Template Editor** которое используется сотрудниками поддержки приложения SLC для заполнения и редактирования правил и шаблонов печати.

* Вебинтерфейс для обслуживания пользователей, загрузки дополнительных данных.test
* Сервер обработки обрабатывающий запросы.

* Подробнее архитектура проекта изложена на [схеме по ссылке](http://portal.gloria-jeans.ru/_layouts/15/DocIdRedir.aspx?ID=7XHH5PPSSUZ5-1518768844-842).

### Процесс формирования ярлыка

Процесс работы системы можно представить в виде четырёх этапов:
* Получение входных и выходных параметров для печати модели на портале http://slc/ от пользователя или другой информационной системы (*Код, Регион, Параметры печати, Формат печати*)
*	Получение необходимых свойств товара из 1С:Справочники
*	Поиск первого исполняемого правила для текущего товара и подстановка соответствующего правилу шаблона
*	Формирование ярлыка по найденному шаблону и вывод в запрашиваемом формате.
